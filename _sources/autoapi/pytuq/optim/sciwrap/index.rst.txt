   .. py:method:: run(nsteps, param_ini, verbose=False)

      An optimization run.

      :param nsteps: Number of optimization steps (unused in scipy wrapper, kept for API compatibility).
      :type nsteps: int
      :param param_ini: Initial position, an 1d array.
      :type param_ini: np.ndarray
      :param verbose: If True, print optimization results. Defaults to False.
      :type verbose: bool, optional

      :returns: Dictionary of results. Keys are 'samples' (history array of parameters), 'objvalues' (history of objective values), 'best' (best parameters array), 'bestobj' (best objective value).
      :rtype: dict
