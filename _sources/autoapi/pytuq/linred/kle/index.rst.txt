      A 2d array of size `(N,K)` consisting of all eigenvectors.

   .. attribute:: weights2

      Working array of integration weights.

      :type: np.ndarray

   .. attribute:: xi

      An array of size `(M,K)` for latent features of KLE.

   .. py:method:: project(data, subtract_mean=True)

      :param data: Data array of size `(N,M)`.
      :type data: np.ndarray
