==========
Plotting
==========

Examples demonstrating plotting utilities for colors, function
visualization, and parameter slicing.


ex_colors.py
------------

Color palette generation and visualization.

Creates and displays a set of RGB color triples using PyTUQ's plotting
utilities, useful for creating consistent color schemes in plots.


ex_plot_fcn.py
--------------

Anchored 1D and 2D function plotting.

Shows how to use plotting utilities to visualize model functions with
respect to one or two parameters while fixing others at nominal values.


ex_slice.py
-----------

2D function slicing and visualization.

Plots 2D slices of functions at various anchor points, useful for
exploring function behavior around specific parameter values.
