compute_stdev(Amat, method='chol')[source]

Computation of pushed-forward standard deviation using a few different methods.

Parameters:

  • Amat (np.ndarray) – A 2d array of size \((N, K)\) each row holding basis evaluations at a training point.

  • method (str, optional) – Method of computation. Options are ‘chol’ (cholesky), ‘choleye’ (cholesky with regularization), ‘svd’ (SVD), ‘loop’ (slow and painful), ‘fullcov’ (slow-ish and memory intensive).

Returns:

An 1d array of pushed-forward standard deviation, of size \(N\).

np.ndarray
