class pytuq.minf.priors.Prior_uniform(infer, domain=None)[source]

Bases: Prior

Uniform prior class.

domain[source]

A 2d array of size (d,2). Can be None, for uninformative priors with no range constraint.

Type:

np.ndarray

factor

A standard deviation factor outside of which the prior returns very low value. For now, we use \(f=3\) for HG, and \(f=\sqrt{3}\) for LU. This is somewhat empiric and should be used with care.

Type:

float

Note

Note that unlike normal prior, this is not a uniform prior on all parameters. This is somewhat empirical rejection mechanism if embedded parameter values lead to values outside given ranges.
