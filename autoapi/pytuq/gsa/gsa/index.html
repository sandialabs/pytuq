pytuq.gsa.gsa.model_sens(model, model_params, domain, method='SamSobol', nsam=100, plot=True, **kwargs)[source]

Sensitivities of a multioutput model.

Parameters:

  • model (callable) – Model of a form \(M(x, p)\) where we are interesting in sensitivities with respect to \(x\), and \(p\) are auxiliary parameters. The model is multioutput, taking \((N,d)\)-sized array, and returning a \((N,o)\)-sized array.

  • model_params (list) – List of model’s auxiliary parameters \(p\).

  • domain (np.ndarray) – Domain of input, a 2d array of size \((d,2)\).

  • method (str, optional) – Sensitivity method. Default is SamSobol. Other options are PCSobol, LinReg or Moat.

  • nsam (int, optional) – Number of samples. Note that for some methods this is not the number of model evaluations needed.

  • plot (bool, optional) – Whether to plot or not.

  • **kwargs – Other keyword arguments for initializing sensitivity objects.

Returns:

Main and total sensitivities arrays, each of size \((o,d)\).

Return type:

tuple(np.ndarray, np.ndarray)
